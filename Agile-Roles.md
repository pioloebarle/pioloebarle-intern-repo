**Reflection**

As a **Mobile Developer**, my primary role is to build and maintain the user-facing mobile application of Focus Bear using the needed tools provided. In the Agile process, I contribute by breaking down mobile-specific features into a testable modules to ensure no bugs found in the mobile app and provide technical estimates during planning to ensure we deliver a functional product incrementally.

My work overlaps significantly with **QA**, as I must ensure my code is testable and error-free before it reaches the end user to maintain a stable app environment. I also collaborate closely with the **Product Manager** to ensure the technical implementation of mobile features aligns with the overall product vision and user needs. Additionally, I interact with backend developers to integrate APIs, ensuring that data flows correctly between the server and the mobile interface.
